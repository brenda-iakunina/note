Erstelle in diesem Ordner eine Datei namens index.html​.

Das Grundgerüst HTML kannst du mit dem ! Kürzel aufbauen.
Füge einen header mit einer Navigation.
Innerhalb der Navigation erstellst du eine ungeordnete Liste, in welcher jeder Listenpunkt einen Hyperlink enthält.
Zusätzlich kannst du ein Bild runterladen und innerhalb eines figure Elements einbinden. Das figure Element ist ein Kind von main.
 
Erstelle in einem Unterordner namens "css" eine style.css.
Vergiss nicht die css Datei in der html Datei zu verknüpfen!

Style nun die verschiedenen Zustände deiner Hyperlinks. Achte auf die Reihenfolge der Zustände!
